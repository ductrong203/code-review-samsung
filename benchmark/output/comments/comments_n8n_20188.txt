`zod.parse()` is a relatively expensive operation involving schema validation. Since `publicUsers` is presumably already validated or comes from a trusted internal source, consider using `zod.safeParse()` or, better yet, a simple object transformation (e.g., `lodash.pick` or object destructuring) to strip the fields instead of re-validating the entire object structure for every item in the list.</note>
