<note>**[Performance]** The use of `Arrays.stream(names).anyMatch(...)` inside `atLeastOneNameMatchesPattern` introduces unnecessary object allocation and stream overhead for every pattern check. 

Since `names` is a small array (typically 1-2 elements in this context), a simple `for` loop is significantly more performant and avoids the overhead of creating a stream pipeline and lambda capture.

**Suggested Fix:**
```java
private static boolean atLeastOneNameMatchesPattern(Object pattern, String[] names) {
    for (String name : names) {
        if (matchesPattern(pattern, name)) {
            return true;
        }
    }
    return false;
}
```</note>
