**[Performance]** The `communicate_with_quorum` function performs a `start_time.elapsed()` call inside the `while` loop. While `Instant::elapsed()` is generally efficient, calling it repeatedly in a hot loop that processes network responses can be optimized by caching the start time or using a monotonic clock reference if the loop duration is expected to be high-frequency.</note>
