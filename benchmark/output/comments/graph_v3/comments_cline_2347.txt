<note>The detectEncoding function uses jschardet.detect(fileBuffer), which returns an object containing 'encoding' and 'confidence'. The current implementation checks for a string return or a property named 'encoding', but it ignores the 'confidence' score. If jschardet returns an encoding with very low confidence, the code will proceed to decode using that potentially incorrect encoding, leading to garbled text.</note>
