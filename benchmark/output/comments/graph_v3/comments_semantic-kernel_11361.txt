<note>**[Code Defect]** The call to agent.InvokeAsync(...).FirstAsync() is unsafe. If the agent returns an empty stream or fails to produce a message, FirstAsync() will throw an InvalidOperationException. Additionally, ChatCompletionAgent.InvokeAsync returns an IAsyncEnumerable<ChatMessageContent>, and calling FirstAsync() without ensuring the stream is not empty is a common source of runtime crashes.

**[Maintainability and Readability]** The hardcoded prompt string is duplicated from line 70. This violates the DRY (Don't Repeat Yourself) principle, making the code harder to maintain if the prompt needs to be updated in both locations.

**[Performance]** The code uses .FirstAsync() on the IAsyncEnumerable returned by agent.InvokeAsync. If the agent returns a stream of multiple messages or chunks, this approach ignores subsequent content, potentially leading to incomplete responses or resource leaks if the underlying stream is not properly disposed of after the first item.</note>
