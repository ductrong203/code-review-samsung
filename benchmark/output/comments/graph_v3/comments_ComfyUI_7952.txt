<note>**[Security Vulnerability]** The RegexMatch node uses user-provided regex patterns without any validation or complexity limits. This allows for ReDoS (Regular Expression Denial of Service) attacks, where a maliciously crafted regex can cause catastrophic backtracking, consuming excessive CPU and hanging the server.

**[Performance]** The regex pattern is not pre-compiled. In ComfyUI, nodes are often executed repeatedly. Compiling the regex inside the execute method causes unnecessary overhead. Use re.compile() or a cache decorator to store compiled patterns.</note>
