<note>**[Code Defect]** The `mp_rand_seed` function returns an uninitialized `mp_rand_state` if `av_random_bytes` fails to fill the entire buffer.

`av_random_bytes` returns the number of bytes read or a negative value on error. If it returns a value greater than 0 but less than `sizeof(ret.v)`, the `ret.v` array will contain partially uninitialized data, leading to non-deterministic PRNG behavior. You should check if the return value equals `sizeof(ret.v)`.</note>
