<note>The use of `is_subclass_of($value::class, \UnitEnum::class)` is redundant and potentially fragile. Since PHP 8.1, all Enums implement the `\UnitEnum` interface. Checking `is_subclass_of` on an object instance is less idiomatic than using `instanceof \UnitEnum`.</note>
